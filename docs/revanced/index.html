GUIDE・YTRevanced/YT+

Un guide complet pour vous accompagner dans l'installation d'un tweak YouTube sur votre smartphone ou tablette.

Temps de lecture :     10 min
Auteurs :    Dhalian

Préambule

YouTube ReVanced et YouTube Plus sont des versions améliorées de YouTube vous permettant de personnaliser votre expérience.
De nombreuses fonctionnalités permettent d'améliorer l'expérience utilisateur comme SponsorBlock qui permet de passer automatiquement les passages sponsorisés.
YouTube ReVanced et YouTube Plus embarquent également un bloqueur de publicités intégré, permettant de regarder une vidéo sans coupure.


Ce guide vous permettra d'installer YouTube ReVanced ou YouTube Plus sur votre appareil.
Ce guide n'est applicable que pour les utilisateurs possédant un appareil Android exécutant Android 8.0 ou supérieur ou possédant un appareil iOS/iPad OS éxécutant iOS 15 ou supérieur.


  • Utilisateurs Android : Suivez les étapes I, II et III.
  • Utilisateurs iOS : Suivez l'étape IV.

I. Installation de YouTube ReVanced

PRÉREQUIS :

  • La dernière version de ReVanced Manager
  • Installez l'APK de "ReVanced Manager". Une confirmation vous sera demandée, appuyez sur "Installer".
  • Rendez-vous dans les paramètres de ReVanced. Puis désactivez "Auto update patches".
  • Appuyez sur "Patcher", puis "Select an application".

  • Tapez "YouTube" dans la barre de recherche. Appuyez sur "Suggested".

    • Note: cela va vous renvoyer vers une requête Google pour obtenir l'APK de la version recommandée de YouTube à patcher.
  • Appuyez sur le premier lien (APKMirror).
  • Appuyez sur "Download APK", vous serez redirigé vers la page de téléchargement. Le téléchargement se lancera automatiquement.
  • Une fois le téléchargement terminé, rendez-vous à nouveau sur "ReVanced Manager", sélectionnez "Patcher" puis "Storage", sélectionnez ensuite l'APK de YouTube que vous venez de télécharger à l'instant.
  • Lancez en appuyant sur "Patch". Patientez en attendant que le processus se termine.

    • Il est recommandé de désactiver votre application YouTube dans les paramètres avant de procéder afin de ne pas confondre les deux applications.

  • Lorsque le processus est terminé (l'option "Install" sera disponible), appuyez sur "Install", il vous sera probablement demandé de laisser "ReVanced Manager" d'installer des applications de source inconnue, donnez l'autorisation à l'application.
  • YouTube ReVanced devrait maintenant s'installer. Patientez le temps que l'application s'installe sur votre appareil.

YouTube ReVanced est désormais installé sur votre appareil. Poursuivez vers l'installation de GMS Services.

II. Installation de GMS Services

PRÉREQUIS :

  • La dernière version de F-Droid

GMS Services est une passerelle essentielle permettant de vous connecter à votre compte Google via YouTube ReVanced.

  • Installez l'APK de "F-Droid". Une confirmation vous sera demandée, appuyez sur "Installer".
  • Ouvrez F-Droid, patientez durant le chargement des dépôts. Allez ensuite dans l'onglet "Paramètres" puis "Dépôts", appuyez sur "+".
  • Appuyez sur "Scanner le QR Code" et scannez le QR Code suivant :
  • Allez dans "Nouveautés", appuyez sur et tapez "microg". Cherchez "microG Services" et installez-le.
  • Rendez-vous dans YouTube ReVanced et choisissez votre compte Google.

Votre compte Google est maintenant utilisable avec YouTube ReVanced.

III. Configuration des téléchargements de vidéos (Optionnel)

PRÉREQUIS :

  • La dernière version de Seal
  • Installez l'APK de "Seal". Une confirmation vous sera demandée, appuyez sur "Installer".
  • Rendez-vous dans YouTube ReVanced.
  • Allez dans "Vous" puis appuyez sur l'icône d'engrenage pour vous rendre dans les paramètres.
  • Appuyez sur "ReVanced" puis "Player" puis "External Downloads".
  • Activez l'option "Show external download button" si elle n'est pas activée.
  • Appuyez ensuite sur "Downloader Package Name".
  • Remplacez la valeur par défaut par com.junkfood.seal et appuyez sur "OK".

Vous pouvez désormais télécharger des vidéos depuis YouTube ReVanced.

IV. Alternative iOS (YouTube Plus)

Cette partie traitera de l'alternative YouTube ReVanced pour les appareils iOS : YouTube Plus.

PRÉREQUIS :

  • SideStore installé et configuré (guide).
  • Activez le profil VPN StosVPN, configuré au préalable dans le guide Sideloading iOS.
  • Sur votre appreil iOS/iPadOS, cliquez sur le bouton ci-dessous pour ajouter le repo Dan's Workshop à SideStore :
  • Rendez vous ensuite dans l'onglet "Browse".
  • Scrollez pour afficher la barre de recherche et cherchez "Youtube" puis cliquez sur le bouton "Free" pour lancer l'installation de YouTube Plus.
