PlayStation 4 - Lumia Guides

Temps de lecture :    > 17 min
Auteurs :    numbersix  Ayhanblackflag

Rejoignez Homebrew France pour obtenir de l'assistance ou nous aider à développer le guide.

Nom : PlayStation 4™ (PS4)

Constructeur : Sony

Année de sortie (EU) : 2013

Préambule

Ce guide permettera de jailbreaker votre PlayStation 4 selon le firmware actuellement installé sur votre console.

I. Sélection du firmware

Rendez vous dans "Paramètres" ➜ "Système" ➜ "Informations sur le système". Vous y trouverez la version de votre firmware. Sélectionnez la version ci-dessous :

  Vérifiez la version du firmware de votre console

Entrez le ensuite ci-dessous pour être redirigé vers le guide approprié.

Vous serez redirigé(e) vers la marche à suivre adéquate selon votre version de firmware.
