PlayStation 4™ - Lumia Guides

Rejoignez Homebrew France pour obtenir de l'assistance ou nous aider à développer le guide.

Préambule

Ce guide vous permettra d'éxecuter le jailbreak PS4 sur le firmware 11.00. Il nécessite qu'une partie de l'exploit soit injectée à l'aide d'une clé USB dédiée, elle sera nécessaire à chaque redémarrage de la console pour refaire le processus.

I. Mise à jour de la console en firmware 11.00

Attention : la section I. n'est à réaliser que si votre console n'est pas DÉJÀ sur le firmware 11.00.

PRÉREQUIS :

  • Une PlayStation 4™ ou une PlayStation 4™ Pro sous le firmware 11.00 ou inférieur.
  • Une clé USB.
  • Un câble Ethernet.
  • Un PC sous Windows.
  • Le firmware 11.00 
  • La dernière version de Rufus 
  • Exécutez "rufus-x-x.exe" téléchargé précédemment.
  • Sélectionnez dans le menu de sélection en haut votre clé USB.
  • Dans "Type de démarrage", sélectionnez "Non-démarrable" puis dans "Système de fichiers", choisissez FAT32.
  • A la racine de votre clé USB, créez un dossier nommé PS4, dans ce même dossier, créez un dossier nommé UPDATE
  • Copiez le fichier du firmware 11.00 téléchargé précedemment dans le dossier UPDATE que vous avez créé.
  • Allumez votre console et déconnectez-la d'Internet par précaution pour qu'elle ne se mette pas à jour automatiquement.
    • Eteignez complètement votre PS4 et appuyez sur le bouton d'alimentation pendant 7 à 8 secondes jusqu'à ce que vous entendez 2 bips.
  • Sélectionnez "Mettre à jour depuis le périphérique de stockage USB".
  • Branchez la clé USB contenant le firmware 11.0 et cliquez sur OK.

II. Configuration sur le PC et la PlayStation 4™

PRÉREQUIS :

  • La dernière version de GoldHEN 
  • La dernière version de PPPwn GUI 
  • Branchez un câble Ethernet entre votre PlayStation 4™ et votre PC. Allez dans "Paramètres" ➜ "Réseau" ➜ "Par câble LAN" ➜ "Personnalisé" ➜ "PPPOE", mettez ce que vous voulez en nom d'utilisateur et mot de passe puis mettez tout en automatique et réglez le Proxy sur "Ne pas utiliser".
  • Mettez le fichier Goldhen.bin à la racine de votre clé USB.
  • Allez sur votre PC, dans "Paramètres" ➜ "Réseau" ➜ "Modifier les paramètres d'adaptateur" ➜ Faites clic-droit sur Wi-Fi ➜ "Propriétés" ➜ "Partage" et cochez "Autoriser les autres utilisateurs à se connecter au réseau de cet ordinateur" ensuite sélectionnez "Ethernet" dans le menu déroulant et cliquez sur "Accepter".
  • Installez PPPwn GUI (téléchargé précédemment) et lancez-le. Une fois lancé, sélectionnez "Ethernet" dans le menu déroulant, cliquez sur les trois petits points et sélectionnez stage2_1100.bin, dans le menu déroulant à droite, sélectionnez "11.0"
  • Sur votre PS4, allez dans "Réseau", et placez vous sur "Tester la connexion Internet", ensuite branchez votre clé USB avec le payload GoldHEN sur la PS4.
  • Appuyez sur "Run exploit" sur votre PC et en même temps "Test la connexion Internet" sur votre PS4.
  • Normalement si vous avez réussi, vous devriez avoir une notification qui montre que le GoldHEN est activé, sinon réessayez.

    • Vous avez terminé le jailbreak de votre PlayStation 4™

III. Alternative (en cas de manque de matériel)

  • Il existe un kit AliExpress pour environ 20€ pour vous simplifier la tâche.
  • Quand vous passez sur votre Wi-Fi une fois jailbreak, mettez les DNS de Nomadic qui sont 62.210.38.117
