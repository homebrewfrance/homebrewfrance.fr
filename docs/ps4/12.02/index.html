GUIDE・PlayStation 4

Préambule

Ce guide explique la préparation d'un Blu-ray BD-J (Lapse.iso) et l'utilisation de GoldHEN via USB ou via le serveur binLoader pour lancer l'exploit BD-J sur PS4 (firmwares 9.00 à 12.02).

PRÉREQUIS :

  • PS4 sur le firmware 9.00 à 12.02 — vérifiez dans Paramètres ➜ Système ➜ Informations système. Si vous êtes au-dessus, attendre un futur exploit.
  • Lecteur Blu-ray : nécessaire pour graver l'ISO sur un disque vierge BD-R (les DVD ne fonctionnent pas).
  • Ordinateur : avec un logiciel de gravure (ex. : ImgBurn sur Windows, K3b sur Linux).
  • Clé USB (optionnelle mais recommandée) : formatée en FAT32 pour charger des payloads comme GoldHEN.
  • Télécharger l'ISO du projet BD-J + Lapse sur le GitHub du développeur Gezine : BD-JB-1250 — choisir Lapse.iso (version offline AIO 1.2 recommandée, inclut fixes pour écrans noirs et corruption de saves).
  • Payload GoldHEN : téléchargez la version compatible avec votre firmware depuis des sources fiables (ex. MediaFire, forums de confiance ou dépôts recommandés par la communauté).
  • Désactiver les mises à jour automatiques : Paramètres ➜ Système ➜ Mises à jour automatiques système ➜ Désactivé.

I. Téléchargement et préparation du Blu-ray BD-JB

Comme indiqué dans les Outils, si ce n'est pas déjà fait téléchargez Lapse.iso (version 1.2 recommandée, inclut les fixes pour bugs).

  • Téléchargez l'outil de gravure (ex. ImgBurn sur Windows si nécessaire).
  • Sélectionnez "Write Image File to Disc".
  • Choisissez l'image ISO Lapse.iso.
  • Insérez un BD-R vierge et gravez à vitesse lente (4x max) pour éviter les erreurs.

Durée estimée : environ 10–20 minutes selon votre graveur.

Vous pouvez acheter un BD-JB déjà prêt mais faites très attention aux arnaques en ligne.

II. Préparation de la clé USB pour le Payload (GoldHEN)

  • Formatez la clé en FAT32 (clic droit ➜ Formater sur Windows ou outil équivalent sur Linux/macOS).
    • Note: Si votre clé fait 64 Go ou plus, utilisez un outil comme guiformat32. Il est souvent plus simple d'utiliser une petite clé (16–32 Go).
  • Créez à la racine un dossier nommé payloads (ou placez directement payload.bin à la racine pour l'auto-chargement).
  • Copiez goldhen.bin renommé en payload.bin dans la clé (la PS4 montera le périphérique comme /mnt/usb0/).
    • Note: Si aucun périphérique USB n'est présent, l'exploit peut lancer un serveur d'écoute sur le port 9020 et vous permettre d'envoyer le payload depuis un PC (méthode plus complexe).

III. Préparation de la PS4

  • Connectez la PS4 à Internet au moins une fois (nécessaire pour activer le lecteur Blu-ray).
  • Paramètres ➜ Notifications ➜ Désactivez "Désactiver les pop-ups pendant la lecture vidéo" (utile pour voir les logs).
  • Éteignez complètement la PS4 (pas en veille).

IV. Lancement de l'exploit

  • Insérez la clé USB (si utilisée) dans la PS4.
  • Insérez le disque gravé dans le lecteur Blu-ray.
  • Allumez la PS4 et allez dans la section Blu-ray (icône disque).
  • Lancez le disque : il se comportera comme un Blu-ray normal et chargera l'exploit BD-J.
  • Attendez 1–2 minutes : des notifications pop-up (logs) apparaîtront. Si succès, Lapse se chaînera automatiquement.
  • Si la clé USB est connectée : elle chargera payload.bin et lancera GoldHEN (l'icône HEN apparaîtra en haut à droite).
  • Sans USB : un serveur binLoader s'ouvrira sur le port 9020. Utilisez un outil sur PC (ex. PS4 Payload Injector) pour envoyer le payload via l'IP de la PS4 (trouvable dans Paramètres ➜ Réseau).
  • Vérifiez le succès : GoldHEN est chargé si vous voyez l'icône. Vous pouvez maintenant lancer des homebrews, cheats, etc.

Durée estimée par tentative : environ 2–5 minutes.
