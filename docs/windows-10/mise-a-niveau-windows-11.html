ESU Windows 10 - Lumia Guides

Préambule

Ce guide vous permettra de mettre à niveau Windows 10 vers Windows 11.

Dans le cas où votre ordinateur n'est pas officiellement compatible avec Windows 11, ce guide sera quand même applicable.

I. Mise à niveau vers Windows 11

Dans cette section, vous apprendez à mettre à niveau Windows 10 vers Windows 11.

Cette section est applicable même si votre ordinateur ne respecte pas les exigences minimales officielles (sauf l’architecture 64 bits, qui reste obligatoire).

Microsoft ne recommande pas la mise à niveau vers Windows 11 sur du matériel incompatible. Certaines fonctionnalités pourraient ne plus fonctionner après un certain nombre de mise à jour.

PRÉREQUIS :

  • L’ISO officiel de Windows 11 (en français)
  • Un PC en 64 bits avec Windows 10 installé et activé
  • Téléchargez l’ISO de Windows 11 depuis le site officiel Microsoft.
  • Faites un clic droit sur le fichier ISO et sélectionnez "Monter".
  • Dans la liste des lecteurs, l’ISO apparaît comme un disque d’installation virtuel.
  • Appuyez sur la touche Windows, recherchez "Invite de commandes", puis exécutez-la en tant qu’administrateur.
  • Dans l’invite de commandes, tapez la commande suivante :
    reg add "HKLM\SYSTEM\Setup\MoSetup" /v AllowUpgradesWithUnsupportedTPMOrCPU /t REG_DWORD /d 1 /f
  • Accédez au disque d’installation monté à l’étape 2. et lancez setup.exe.
  • Poursuivez les étapes d’installation et sélectionnez l’option "Conserver les fichiers personnels et applications".

Votre ordinateur sera ensuite mis à niveau vers Windows 11.
