ESU Windows 10 - Lumia Guides

Rejoignez Homebrew France pour obtenir de l'assistance ou nous aider à développer le guide.

Préambule

Dans ce guide nous allons voir les différents moyens de garder Windows 10 tout en recevant encore des mises à jour de sécurité.

Encore une fois, plusieurs options s'offrent à vous :

  • Option 1 : permet de conserver votre installation Windows 10 à jour jusqu'octobre 2026 en quelques clics. (voir Section I.)
  • Option 2 : permet de mettre à niveau vers la version Entreprise IoT de Windows 10 pour recevoir des mises à jour de sécurité jusque 2032 (nécessite un peu de temps). (voir Section II.)

I. Activation du support étendu gratuit.

Cette section vous permettra de bénéficier d'un support étendu d'un an pour votre installation actuelle de Windows 10, jusque octobre 2026.

  • Rendez vous dans les Paramètres de Windows 10.
  • Allez dans "Mise à jour et sécurité", ouvrez Windows Update et veillez à être sur la version 22H2, si ce n'est pas le cas, effectuez la mise à jour.
  • Retournez à l'accueil des Paramètres et sélectionnez "Comptes" puis "Sauvegarde Windows", il ne vous reste plus qu'à cocher les options de synchronisation souhaitées, les paramètres sélectionnés seront synchronisés avec OneDrive.

L'activation du support étendu (ESU) ne se fera pas de façon instantanée, vous serez mis sur liste d'attente.

II. Mise à niveau vers une version Entreprise LTSC

Cette section vous permettra de bénéficier d'un support étendu de 7 ans pour Windows 10 en changeant d'édition, jusque janvier 2032.

PRÉREQUIS :

  • L'ISO de Windows 10 IoT Entreprise LTSC 2021
  • Dans l'Explorateur de Fichiers faites clic-droit sur le fichier ISO de Windows 10 IoT Entreprise LTSC puis sélectionnez "Monter".
  • Dans la liste des lecteurs, le fichier ISO apparaît comme un disque d'installation.
  • Appuyez sur la touche Windows et recherchez "Invite de Commandes", une fois que vous l'avez trouvé faites "Exécuter en tant qu'administrateur".
  • Dans l'invite de commandes tapez la commande suivant :
    reg add "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion" /v EditionID /d IoTEnterpriseS /f
  • Accédez rapidement au disque d'installation que vous avez monté dans l'étape 1. et lancez setup.exe
  • Poursuivez l'étape de l'installation et assurez-vous d'avoir l'option "Keep personal files and apps", si c'est le cas, cliquez-dessus et poursuivez.

    • Votre édition de Windows 10 sera ensuite mise à niveau vers Windows 10 IoT Entreprise LTSC 2021.
