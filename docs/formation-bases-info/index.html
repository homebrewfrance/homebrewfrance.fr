Formation des bases de Windows - HomebrewFrance Docs

Rejoignez Homebrew France pour obtenir de l'assistance ou nous aider à développer le guide.

Préambule

Cette courte formation vous permettra d'acquérir les connaissances de base et de maîtriser vôtre ordinateur sous Windows.

Cette formation ne traite pas des éditions avancées (Education, Entreprise ou Serveur).

Liste des versions supportées par cette formation :
  • Windows 7 (2009) : éditions Starter, Familiale, Familiale Premium, Professionel, Ultimate.
  • Windows 8 (2012) : éditions Basique, Professionel.
  • Windows 8.1 (2013) : éditions Basique, Professionel.
  • Windows 10 (2015) : éditions Basique, Professionel.
  • Windows 11 (2021) : éditions Basique, Professionel.

I. Identifier votre version de Windows

Afin d'identifier votre version de Windows, vous pouvez en permier lieu identifier l'apparence de la barre des tâches, la barre des tâches est l'élement qui se situe en bas de votre écran lorsque votre ordinateur est allumé et que vous vous êtres connecté à votre compte utilisateur.
